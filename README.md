## Общие сведения:

---

В данном репозитории содержатся лабораторные работы, выполненные в рамках курса. Каждая лабораторная работа оформлена в виде Jupyter Notebook, в котором содержится код, комментарии и результаты выполнения с графиками.

В каждой лабораторной работе реализуются различные модели машинного обучения для решения задач классификации и регрессии на соответствующих датасетах, взятых с kaggle. В ходе работ сначала реализуются базовые модели с помощью библиотеки `sklearn`, их улучшеные версии с препроцессингом данных и подбором гиперпараметров, а после создаются собственные реализации моделей с нуля.

---

Датасеты взяты с платформы [Kaggle](https://www.kaggle.com/):

**Датасет для задачи классификации:** [Water Quality](https://www.kaggle.com/datasets/adityakadiwal/water-potability)

*В задаче классификации необходимо классифицировать воду на пригодную и непригодную для питья на основе различных химических компонентов.*

**Датасет для задачи регрессии:** [Revisiting a Concrete Strength regression](https://www.kaggle.com/datasets/maajdl/yeh-concret-data)

*В задаче регрессии необходимо предсказать прочность бетонной смеси на основе ее состава и возраста.*
